# Project Summary - PredictMyFight

## Overview

PredictMyFight is a comprehensive final year thesis project that demonstrates the application of statistical analysis and weighted algorithms to predict Mixed Martial Arts (MMA) fight outcomes. The project showcases modern web development practices, data visualization techniques, and algorithmic problem-solving.
